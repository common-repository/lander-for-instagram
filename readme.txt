== Description ==

Landing page featuring a mirror of your instagram feed with links to related posts

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/lander-for-instagram
2. Go to Settings and add your page access token and instgram business account id:
https://developers.facebook.com/docs/instagram-api/getting-started/
3. Add the [instagram_lander] short code where you want it to be displayed
